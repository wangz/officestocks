# officestocks
for check stocks in office 办公室看股软件

#安装

1.安装python

2.安装wxpython

可以去官网下载，找不到的话可从这里下载：http://pan.baidu.com/s/1qXbznWS

3.双击oss.pyw即可，点击go,最小化后标题显示数字为股票指数

效果图：https://github.com/wangz/officestocks/wiki
